Code knowledge graph MCP server. AST indexing via tree-sitter for 15+ languages.

$ npm i -g @levnikolaevich/hex-graph-mcp Copy

Tools

7 MCP tools for code knowledge graph operations.

ToolDescriptionKey Feature

search_symbols Full-text search for symbols by name FTS5, filter by kind

trace_calls Trace call chains (callers or callees) BFS with configurable depth
get_context 360-degree view of a symbol Definition, callers, callees, siblings
get_architecture Project architecture overview Module matrix, hotspots
watch_project File watcher for incremental graph updates Singleton, CASCADE cleanup on delete

Architecture

  • Storage: SQLite via better-sqlite3, single .hex-graph.db per project
  • Parsing: tree-sitter WASM — 15+ languages (JS, TS, Python, Go, Rust, Java, C, C++, C#, Ruby, PHP, Kotlin, Swift, Bash)
  • Indexer: 4-pass pipeline — PURGE → SCAN → PARSE → RESOLVE
  • Watcher: chokidar for incremental updates, CASCADE cleanup on delete
