Hooks & Output Style

Hooks install via setup_hooks tool. One call configures everything — PreToolUse blocking, PostToolUse filtering, and Output Style.

EventTriggerAction
PreToolUse Read/Edit/Write/Grep on text files Blocks built-in, forces hex-line tools
PreToolUse Bash with dangerous commands Blocks rm -rf /, git push --force, DROP, chmod 777
PostToolUse Bash output exceeding 50 lines RTK: normalize, deduplicate, truncate (first 12 + last 12)
SessionStart Session begins Injects tool preference list into agent context

Output Style Hack

A persistent Output Style installed to ~/.claude/output-styles/hex-line.md

Embeds tool preferences directly in Claude's system prompt — reducing hook firings by making Claude prefer hex-line tools from the start.

Installed automatically by setup_hooks. Activates if no other output style is set.

Hash Format

Every line is annotated with a 2-char content hash and line number.

ab.42    const x = calculateTotal(items);

ab — 2-char FNV-1a tag from content (whitespace-normalized)

42 — line number (1-indexed)

Range checksums: checksum: 1-50:f7e2a1b0 — FNV-1a accumulator, detects any change in the range.
